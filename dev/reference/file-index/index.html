    localpass Complete File Index¶

    This file index contains a comprehensive directory of every .py, .js, .css, .html, and .json file in the localpass repository. Files are sorted alphabetically by directory and then by filename within each directory.

    File Path Type Lines Purpose Key Functions/Classes
    generate_docs.py Python Source 1057 Offline developer documentation generation script. Parses source files to build layout templates. main(), DocGenerator
    localpass/init.py Python Source 1 Package initializer None
    localpass/main.py Python Source 40 Primary CLI entry point. Initializes local configurations, boots the TUI, and starts the background local server thread. main()
    localpass-extension/background.js JavaScript 431 Extension background service worker. Handles unified communication with local Python host (native messaging or localhost HTTP fallback). Caches tokens and orchestrates WebAuthn dialogue windows. nativeCall(), httpCall(), chrome.runtime.onMessage.addListener()
    localpass-extension/content.js JavaScript 685 Content script injected into webpages. Orchestrates MutationObserver field observers, handles login input detection, button injection, and autocomplete dropdowns. injectNLButtons(), showInlineDropdown(), removeDropdown()
    localpass-extension/manifest.json JSON Configuration 63 Extension manifest file defining metadata, execution permissions, scripts, offline service workers, and CSP policies. None
    localpass-extension/popup/download_phosphor.py Python Source 22 Utility script to fetch and cache Phosphor web icon assets locally for offline extension usage. download_icons()
    localpass-extension/popup/passkey_dialog.html HTML Document 244 Window layout for WebAuthn passkey registration and signing prompts, styled to match the extension visual theme. None
    localpass-extension/popup/passkey_dialog.js JavaScript 147 Logic for the passkey overlay window. Handles UI events and sends user confirmations back to the background worker. onReady(), confirmPasskey(), sendBg()
    localpass-extension/popup/popup.css CSS Stylesheet 1675 Unified styles for the extension popup and dialog panels, declaring HSL variables, transitions, animations, and dark/light themes. None
    localpass-extension/popup/popup.html HTML Document 226 Main popup popup interface shell. Establishes the layout structure for dynamic Single Page Application (SPA) views. None
    localpass-extension/popup/popup.js JavaScript 3465 SPA controller for the extension popup, rendering dynamic views, filters, generator, and configuration settings. navigateTo(), registerView(), sendBg(), makeEntryRow()
    localpass-extension/popup/theme-init.js JavaScript 22 Bootstrap script executed immediately on popup load to check and apply cached dark/light themes prior to rendering. None
    localpass-extension/utils/bridge.js JavaScript 15 Bridge utilities managing direct port connections between isolated extension contexts and webpage scopes. connectBridge()
    localpass-extension/utils/detector.js JavaScript 114 Login forms and field detection algorithms. Injected alongside content scripts. isFieldVisible(), findUsernameField(), findPasswordField(), findLoginForm()
    localpass-extension/utils/filler.js JavaScript 68 Framework-compatible value insertion layer, bypassing synthetic constraints on React, Vue, and Angular pages. fillField(), fillLoginForm(), fillOTPForm()
    localpass-extension/utils/generator.js JavaScript 259 Dynamic client-side password generation library, producing strong keys matching user character configurations. generatePassword(), checkStrength()
    localpass-extension/utils/webauthn_interceptor.js JavaScript 166 WebAuthn interception worker. Runs in MAIN world at document_start to override native browser APIs. interceptWebAuthn()
    localpass/core/init.py Python Source 1 Package initializer None
    localpass/core/adapter.py Python Source 349 decoupling interface mapping TUI components and HTTP daemon requests directly to Vault payload records. VaultAdapter, find_by_domain(), create_entry(), update_entry(), delete_entry()
    localpass/core/audit.py Python Source 248 Structured local transaction audit service, tracking creation, deletion, reveal, and clipboard copying events. AuditService, AuditEvent, log()
    localpass/core/auth.py Python Source 264 Session coordinator, secure memory allocation context manager, and Argon2id derived key generator. derive_key(), zero_bytes(), SecureBuffer, SessionManager
    localpass/core/domain_trust.py Python Source 270 Phishing-resistant URL domain matching engine, implementing Public Suffix List (PSL) extraction algorithms. DomainTrustService, MatchLevel, MatchResult, _extract_registered_domain()
    localpass/core/entries.py Python Source 208 Data models defining vault payload schemas, including secure LoginEntry, PasskeyEntry, and NoteEntry. Entry, LoginEntry, PasskeyEntry, NoteEntry, VaultPayload, VaultMetadata
    localpass/core/generator.py Python Source 102 Cryptographically secure server-side random string generator, leveraging system entropy sources. generate_password(), check_password_strength()
    localpass/core/migrations.py Python Source 128 Database migration service upgrading older vault files to current schema formats. MigrationManager, migrate_v1_to_v2(), migrate_v2_to_v3()
    localpass/core/passkey.py Python Source 275 Low-level WebAuthn/FIDO2 credentials service, creating CBOR structures and assertions. PasskeyService, create_credential(), get_assertion()
    localpass/core/recovery.py Python Source 172 Recovery service for parsing damaged, partially corrupted, or unclosed vault JSON envelopes. RecoveryManager, forensic_extract(), restore_from_bak()
    localpass/core/totp.py Python Source 34 Multi-step Time-based One-Time Password generator utilizing PyOTP to calculate codes and time remaining. get_totp_info()
    localpass/core/unlock_providers.py Python Source 129 Integrations mapping system keychains, biometric authentications, or local PIN systems to vault keys. UnlockProviderService, PINProvider, BiometricProvider
    localpass/core/vault.py Python Source 421 Encrypted file serializer. Manages atomic writes, legacy b'NLLK' upgrades, and automated local backup rotation. load_vault(), save_vault(), init_vault(), _write_atomic(), _rotate_backups()
    localpass/native_host/init.py Python Source 3 Package initializer None
    localpass/native_host/host.py Python Source 290 Native Messaging runtime. Formats binary length-prefixed standard input streams to process JSON exchanges with Service Workers. NativeHost, read_message(), write_message(), handle_request()
    localpass/native_host/manifest_installer.py Python Source 154 Installer script adding appropriate registry configuration entries to link Chrome/Firefox extension hosts. ManifestInstaller, install_windows_registry(), create_host_manifest()
    localpass/ui/init.py Python Source 1 Package initializer None
    localpass/ui/app.py Python Source 179 UI controller wrapper using prompt_toolkit. Orchestrates views, stack switches, and global keyboard bindings. localpassTUI, run(), switch_screen(), global_key_bindings()
    localpass/ui/components/init.py Python Source 1 Package initializer None
    localpass/ui/components/entry_list.py Python Source 67 Scrollable vertical grid listing search results and details within the main dashboard panel. EntryListSelector, render_entries(), handle_selection()
    localpass/ui/components/layout.py Python Source 109 Layout renderer providing borders, headers, and quick-action menu footers throughout the TUI. TUIDrawer, draw_header(), draw_footer(), wrap_in_border()
    localpass/ui/components/totp_display.py Python Source 35 Animated, periodically-refreshing visual bar displaying 2FA codes and real-time expiration progress. TUITOTPDisplay, render(), update_tick()
    localpass/ui/screens/init.py Python Source 1 Package initializer None
    localpass/ui/screens/dashboard.py Python Source 110 Dashboard view for TUI. Lists entries, handles quick actions, and manages visual item navigation. DashboardScreen, handle_key(), draw()
    localpass/ui/screens/entry_edit.py Python Source 202 Interactive editing form managing changes to login fields, secrets, custom URLs, and secure notes. EntryEditScreen, build_form(), save_changes(), draw()
    localpass/ui/screens/entry_new.py Python Source 213 Screen form assisting creation of new credentials, validating secrets, and saving to disk. EntryNewScreen, build_form(), submit_entry(), draw()
    localpass/ui/screens/entry_view.py Python Source 136 Secure screen displaying usernames, revealing passwords, showing active TOTPs, and offering copy shortcuts. EntryViewScreen, reveal_password(), copy_field(), draw()
    localpass/ui/screens/generator.py Python Source 331 Interactive password generation assistant allowing customization of lengths and character exclusions. GeneratorScreen, refresh_password(), update_options(), draw()
    localpass/ui/screens/search.py Python Source 69 Instant filtering screen matching titles, users, or domains to display list selections. SearchScreen, on_search_change(), draw()
    localpass/ui/screens/settings.py Python Source 219 System options screen adjusting locking schedules, backup retention values, and extension linkages. SettingsScreen, save_settings(), install_extensions(), draw()
    localpass/ui/screens/unlock.py Python Source 253 Boot authentication screen checking the Master Password and initializing core providers. UnlockScreen, handle_unlock(), draw_ui(), draw_lockout()
    localpass/utils/init.py Python Source 1 Package initializer None
    localpass/utils/clipboard.py Python Source 21 OS clipboard interface scheduler initiating secure, background-threaded automatic clipboard wringer timeouts. copy_to_clipboard(), _clear_clipboard_worker()
    localpass/utils/config.py Python Source 90 Configuration database parser saving system layouts, local server ports, and preferences. ConfigManager, load_config(), save_config()
    localpass/utils/paths.py Python Source 76 Local filesystem directory mapper locating the core config, backup folder, vault file, and generating system fingerprints. get_vault_path(), get_backup_dir(), get_device_id(), get_app_dir()
